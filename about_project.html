About the Project

Our diabetes prediction project utilizes a Support Vector Machine (SVM) model to predict the likelihood of diabetes based on various medical parameters. Here's an overview of the project:

Data Handling


The project begins by loading a diabetes dataset, which contains several medical parameters such as glucose levels, blood pressure, and body mass index. The data is split into training and testing sets to evaluate the performance of the model.

Support Vector Machine (SVM) Model


We employ an SVM model, a powerful supervised machine learning algorithm suitable for classification tasks. SVM works by finding the optimal hyperplane that separates the data into different classes. The model's hyperparameters, including kernel type, regularization parameter (C), and gamma, can be tuned to improve performance.From svc model i got an accuracy of 88.9 % which is a good accuracy.
